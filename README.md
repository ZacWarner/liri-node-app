# liri-node-app

### It's like siri but takes node commans

##### Commands
*Concert-this:
    returns concert's for given artist
*Spotify-this:
    Will give you information about a song.
*movie-this:
    gives information about a specific movie title.
*do-what-it-says:
    will pull directions from random.txt
